# Estudio Creativo Web

Este proyecto es un sitio web estático para un **estudio creativo**, desarrollado con **HTML y CSS**.  
El objetivo es ofrecer una presencia online moderna, atractiva y didáctica, mostrando el portfolio, presupuesto, contacto y aviso legal de forma clara y profesional.

## Cómo usar
1. Clona este repositorio.
2. Abre `index.html` en tu navegador.
3. Explora las secciones y el diseño.

## Estructura del proyecto
mi-estudio-creativo/
├── index.html          # Página principal
├── productos.html      # Portfolio con galería de proyectos
├── presupuesto.html    # Formulario de presupuesto
├── contacto.html       # Datos de contacto y mapa
